分析对话历史，生成一段自然语言总结。

## 总结要求
- 用自然语言概括对话的主要内容
- 突出关键话题、用户意图、重要信息
- 控制在 100-200 字
- 不提取具体事件细节（由 memory_event_extractor 处理）
- 不输出 JSON 格式，只输出纯文本段落

## 总结范围
- 对话的主要话题
- 用户的核心意图
- 重要的共识或约定（简要提及）
- 情绪基调变化

## 输出格式
直接输出一段自然语言总结，不要 JSON，不要代码块，不要前后缀。

示例输出：
本次对话主要围绕用户的工作计划展开。用户提到了明天需要完成的项目报告，表达了对截止日期的担忧。AI提供了时间管理建议，双方讨论了可能的解决方案。对话结束时用户表示会尝试建议的方法。

对话：
{recent_msg_list}
